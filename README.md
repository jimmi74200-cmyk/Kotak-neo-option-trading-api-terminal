🚀 Kotak Neo Index Option Trading Dashboard (v1.0)
This high-speed trading dashboard is built with the Kotak Neo API, specifically designed for index options traders for placing orders through API.

📖 How to Install & Run
Follow these simple steps to get started:
Download: Click the Latest Release Link and download the Kotak_Neo_Terminal.zip file.
Extract: Right-click the downloaded ZIP file and select Extract All. Do not run the app from inside the ZIP folder.
Launch: Double-click the Kotak_Neo_Terminal.exe file.
Note: If Windows shows a "Protected your PC" popup, click More Info and then Run Anyway.

✨ Key Features
Dual-Side Execution: Dedicated panels for Call (CE) and Put (PE) with real-time LTP display and instant Buy/Sell buttons.
Smart Strike Selection: Select your Index (Nifty/BankNifty) and Expiry. Use Auto Select to instantly fetch ATM, ITM, or OTM strikes based on the current Spot/Future price.
Integrated Risk Management:
Set default Quantity and Stop Loss (SL) (Default: 10 points).
Zero-SL Mode: Enter 0 in the SL field to place orders without an automated stop-loss.
Pending Order Tracking: Orders placed with a Stop Loss will automatically appear in the Pending Orders section.
Safe Exit Suite: Use Exit at Market or Auto Target to square off positions safely. These buttons are programmed to cancel your pending SL order automatically to prevent accidental "double-fills."
🔐 Secure Login & Privacy
Optional "Remember Login": Use the "Remember" checkbox to save time, or leave it unchecked to enter your keys manually every time.
Local Encryption: If you choose to remember your login, the app generates a locally encrypted .env file on your machine. Your keys are never sent to the cloud or stored within the main application (.exe).
Total Control: If you delete that .env file stored on your pc, the app "forgets" your keys immediately has no access to your credentials. 

⚠️ Critical Usage Instructions
Stop-Loss Workflow: If you place an order with an SL, do not use the Buy/Sell buttons to exit. Instead, use the Exit at Market or Auto Target buttons to ensure your SL order is cancelled correctly.
Manual Exit: If SL is set to 0, you can use the Buy/Sell buttons normally for both entry and exit.
Safety First: Always test the dashboard with the smallest possible quantity (1 lot) to familiarize yourself with the execution speed before placing large orders.
